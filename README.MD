# 🧮 Mi Contador – Prueba Práctica para Practicante

Aplicación web interactiva de contador desarrollada con **HTML5**, **CSS3** y **JavaScript** puro. Permite incrementar, decrementar y resetear un valor numérico mostrado en pantalla, con una interfaz moderna, responsive y efectos visuales atractivos.


## 📋 Requisitos de la prueba

| Requerimiento | Estado |
|--------------|--------|
| Estructura HTML5 semántica | ✅ |
| Título de página "Prueba Practicante - Contador" | ✅ |
| Enlace a CSS y JS externos | ✅ |
| Contenedor principal centrado con Flexbox | ✅ |
| Valor inicial "0" resaltado (tamaño grande, negrita, color) | ✅ |
| Tres botones: Decrementar, Resetear, Incrementar | ✅ |
| Botones con colores distintivos (rojo, gris, verde) | ✅ |
| Lógica JavaScript: incrementar, decrementar, resetear | ✅ |
| Código sin errores en consola | ✅ |
| Efectos visuales hover/active (iluminación, presión) | ✅ |

## 🛠️ Tecnologías utilizadas

- **HTML5** – Estructura semántica con etiquetas `main`, `h1`, `div`, `button`.
- **CSS3** – Flexbox para centrado, variables implícitas, transiciones, sombras y efectos de `hover`/`active`.
- **JavaScript (ES6)** – Manipulación del DOM, eventos `click` con `addEventListener` y funciones flecha.

## 📁 Estructura del proyecto

prueba-practicante-contador/

├── index.html # Documento HTML principal

├── style.css # Estilos y efectos visuales

├── script.js # Lógica del contador

└── README.md # Documentación del proyecto

## 🚀 Cómo ejecutar el proyecto

1. **Clonar o descargar** este repositorio.
2. Abrir la carpeta del proyecto.
3. Hacer doble clic en `index.html` **o** abrirlo con **Live Server** en VS Code.

El contador se abrirá en el navegador predeterminado y estará listo para usarse.
